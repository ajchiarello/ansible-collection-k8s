Role Name
=========

Role to handle Nginx with Ansible Operator SDK

Requirements
------------

None

Role Variables
--------------

- See [`defaults/main/main.yml`](defaults/main/main.yml)
- See [`defaults/main/nginx.yml`](defaults/main/postgres.yml)

Dependencies
------------

collections:
- operator_sdk.util

Example Playbook
----------------

```yaml
- hosts: localhost
  gather_facts: no

    - community.kubernetes
    - operator_sdk.util
  tasks:
    - import_role:
        name: "nginx"
```

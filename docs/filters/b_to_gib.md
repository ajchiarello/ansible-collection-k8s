# b_to_gib
Bytes to GiB

## Synopsis

Convert bytes to GiB




## Parameters

  _input (True, int, None)
    Bytes



## Examples

```yaml
    
      total_gib: "{{ 1024000 | b_to_gib }}"

```


## Return Values

  _value (, float, )
    GiB\.
